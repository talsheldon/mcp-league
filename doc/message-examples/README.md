# Message Examples

This directory contains example JSON messages for the league protocol.

## Example Files

- `referee_register.json` - Referee registration request
- `player_register.json` - Player registration request
- `game_invitation.json` - Game invitation message
- `choose_parity.json` - Parity choice request/response
- `game_over.json` - Game completion message
- `match_result.json` - Match result report

See `protocol-spec.md` for complete message specifications.
